Read the following instruction to run the program :

1. First install bio++ library from this link http://biopp.univ-montp2.fr/wiki/index.php/Installation.
2. Use the following code to compile program from terminal 
	g++ -std=c++11 bio_test.cpp -g -o biotest --static  -I /home/avijit/include -L /home/avijit/lib  -lbpp-phyl -lbpp-seq -lbpp-core
3. Then following code to run the program
	./biotest
